> קו מוצרים חדש המבוסס על מתודיקה אחת: **סוכן AI מתוזמן שמנטר משימה אחת ומגיש דייג'סט/פעולה** — דרך הבוט (וואטסאפ/טלגרם/מייל) שכבר בנינו.

> עברית-first · מקומי (Ollama) לנפח + Claude לאיכות · תאריך: 2026-07-18 · סטטוס: אפיון.

> 🏗️ **סטטוס:** מנוע ה-Agent OS **נבנה בקוד** ב-`helix-ops` (migration-v12 + `lib/agentos` + scheduler + UI, typecheck נקי). מדריך לקוח: `HELIX-DAILY-CUSTOMER-GUIDE`. תשתית מלאה: `OLLAMA-DAILY-DIGEST-INFRA` §8.5.

## 0. התובנה המכוננת — מ-reactive ל-proactive
כל המוצרים הקיימים הם **dashboard שנכנסים אליו**. המתודיקה הזו הופכת כל פונקציה עסקית ל**עובד-AI מתוזמן** ששולח **דייג'סט יומי בבוקר** — "מה שאתה רוצה שיחכה לך בלי לבדוק בעצמך". את שכבת ההגשה (הבוט הרב-ערוצי) כבר בנינו; זה מחבר מיידי.

**המנוע אחד (HELIX Agent OS):** `scheduler → folder-of-agents → digest דרך הבוט`. כל מוצר למטה = **agent pack** על אותו מנוע. בונים פעם אחת (ראה מסמך "תשתית OLLAMA ל-Daily Digest").

**מנוף עלות:** משימות נפח (סיווג/relevance/סנטימנט) על **Ollama מקומי** (אפס עלות API); Claude לאיכות (כתיבה/baldiga). פותר את סיכון "עלות AI" בכל האפיונים.

## 1.5 Skills Registry — ספריית skills להרצה ע"י הסוכנים 🆕
ה-registry (`lib/agentos/registry.ts`) יודע לטעון **חבילות skills בפורמט `SKILL.md`** (frontmatter `reads/writes/depends-on/triggers/env_vars` + גוף-prompt) ולהריץ אותן מול Model Router (Ollama/Claude). המקור הראשון: **Marketing Skills Pack — 64 skills** שנקצרו (MIT) מ-`MoizIbnYousaf/marketing-cli` ל-`helix-ops/skills-marketing/` (4 שכבות: Foundation·Strategy·Execution·Distribution). ה-`cmo` הוא **skill-orchestrator** (בוחר ומריץ skills לפי מטרה) — בלופרינט ישיר ל-agent-pack שיווקי. הפלט עובר `lib/hebrew.ts` לעברית + proofread. זה נותן ל-Agent OS מיד **מאגר יכולות רחב** בלי לכתוב כל skill מאפס, ומאמת את ארכיטקטורת ה-"agent טוען skill → מריץ → מוסר בערוץ".

## 2. מוצרים חדשים (מעבר לשיווק) — כל אחד = agent pack

### 💰 פיננסים / ראיית חשבון
**HELIX Books** ⭐ — עובד-AI לפיננסים של העסק.
- **סוכנים:** מנטר פיד בנק → מסווג הוצאות; מזהה חריגות; **תזכורות מע"מ/מקדמות מס** (ישראלי); רודף חשבוניות שלא שולמו; brief תזרים.
- **דייג'סט:** "תזרים: +₪12k החודש · מע"מ עד ה-15 · 3 חשבוניות באיחור (₪8k)".
- **קהל:** SMB/עצמאים ישראלים. **כאב ענק, לא-שיווקי, TAM רחב.**

**Invoice Chaser** — רודף אוטומטית חשבוניות פתוחות בוואטסאפ (תזכורת מנומסת → הסלמה).

### ⚙️ תפעול
**HELIX Ops Brief** — סוכני מלאי-נמוך, סטטוס הזמנות, בעיות משלוח, תזכורות ספקים → brief תפעולי יומי.

**HELIX Sentinel** ⭐ — שומר דדליינים/חידושים: רישיונות, ביטוחים, חוזים שפגים, דומיינים, **מכרזים ממשלתיים** → התראה מבעוד מועד. "עסק לא מפספס דדליין קריטי."

### 📅 מזכירות / אדמין
**HELIX Desk** ⭐ — מזכירה אוטומטית.
- **סוכנים:** brief לוח-שנה בבוקר; **הכנה לפגישות** (מחקר על המשתתפים); triage/סיכום מיילים; תזכורות ומעקבים; תיאום פגישות בוואטסאפ.
- **דייג'סט:** "3 פגישות היום · הכנתי תקציר על הלקוח ב-10:00 · 2 מיילים דורשים תשובה · follow-up ל-X".

**Meeting Notes** — מצטרף/מקבל תמלול → סיכום + action items → מפיץ למשתתפים.

### 👤 עוזר אישי
**HELIX Daily** ⭐ (ה-hub האופקי) — brief אישי מותאם: לוח-שנה, מזג-אוויר, חדשות בתחום, מיילים חשובים, to-dos, + כל agent שהמשתמש מוסיף. **כל מוצר אחר יכול להיות agent בתוך ה-Daily** → cross-sell מובנה לכל האקו-סיסטם.

**Personal CRM** — מזכיר לחזור לאנשי-קשר, ימי-הולדת, follow-ups שנשכחו.

### 🧑‍💼 HR / גיוס (מתחבר ל-PLUG)
**Recruiting Radar** — מועמדים חדשים תואמי-משרה, תזכורות ראיונות, checklist onboarding → דייג'סט למגייס.

### ⚖️ משפטי / רגולציה
**Compliance Watch** — שינויי רגולציה רלוונטיים, פקיעת חוזים, מעקב תיקים/מועדים.

### 🤝 Customer Success
**Churn Radar** — לקוחות בסיכון-נטישה מסומנים יומית (ירידת שימוש/סנטימנט) + דייג'סט feedback.

### 📊 מודיעין שוק (חוצה-תחום)
**Competitor Watch** — מחירים, השקות, גיוסי-עובדים, חדשות של מתחרים → דייג'סט תחרותי.
**Grants & Tenders Radar** — מכרזים ממשלתיים + קרנות מימון (ישראלי, יקר-ערך) תואמי-פרופיל.

## 4. מונטיזציה (משותפת לקו)
- **מנוי per-workspace** בעברית, מחיר SMB ישראלי (~₪49-199/agent pack, או חבילה).
- **HELIX Daily** = wedge זול/ויראלי → upsell ל-agent packs מקצועיים (Books/Desk/Sentinel).
- **מנוף Ollama** מוריד עלות שוליים → מרווח בריא גם במחיר נמוך.
- **מודל דואלי:** מכירה ישירה תחת מותג HELIX (ערוץ ראשי) + הפצה/שת"פ white-label דרך שותף (ערוץ אופציונלי). כל pack מזין את האקו-סיסטם.

## 5. סיכונים
- **הזיות מודל מקומי** — Ollama קטן טועה; משימות קריטיות (כסף/משפטי) דורשות verification + Claude fallback + human-in-the-loop.
- **גישה לדאטה** — פיד בנק/מייל/לוח-שנה דורשים OAuth/הרשאות רגישות; פרטיות-first (זווית Ollama מקומי = יתרון מכירה).
- **פיזור מיקוד** — הרבה packs; להתחיל ב-3 הכוכבים (Daily/Books/Desk) ולהרחיב לפי ביקוש.

---
**סיכום:** לא "עוד מוצר" — **פלטפורמת עובדי-AI מתוזמנים** על מנוע אחד (Agent OS). כל פונקציה עסקית — שיווק, פיננסים, תפעול, מזכירות, HR — הופכת ל-agent pack ששולח דייג'סט יומי בעברית לוואטסאפ. HELIX Daily הוא ה-hub; Books/Desk/Sentinel הם הכוכבים הלא-שיווקיים עם TAM רחב. הכל על התשתית שכבר בנינו + Ollama להוזלת עלות.
